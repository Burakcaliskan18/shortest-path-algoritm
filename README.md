# Shortest Path Finder Algorithm 

A foundational Java application that calculates the shortest path between cities using Dijkstra's Algorithm.

##  Project Overview
This project was developed during my first year of computer engineering studies. Instead of relying on advanced, built-in data structures, it was built from scratch using core programming fundamentals to strengthen my understanding of algorithm logic.

The primary focus of this application is the practical implementation of **Object-Oriented Programming (OOP)** principles.

##  Key Features
- **Strict OOP Principles:** Focuses heavily on Object-Oriented Programming. Locations and distances are modeled purely through basic classes, objects, and encapsulation, rather than advanced graph theory.
- **Algorithmic Logic:** The entire Dijkstra algorithm logic was built from scratch utilizing only primitive arrays and loops to solidify core programming foundations.

##  Technologies Used
- **Language:** Java
- **IDE:** IntelliJ IDEA
